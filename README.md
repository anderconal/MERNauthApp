# MERNauthApp

From Advanced React and Redux Udemy's course by Stephen Grider.

###Getting Started###

####Familiar with Git?#####
Checkout this repo:

```
	> git clone https://github.com/anderconal/MERNauthApp.git
```

###Install dependencies###

```
	> cd MERNauthApp
  	> npm install
  	> cd client
  	> npm install 
```
###Start MongoDB###
```
	> mongod
```

###Start the server###

```
	> cd MERNauthApp
  	> npm run dev
```

###Start the client###

```
	> cd client
  	> npm start
```
